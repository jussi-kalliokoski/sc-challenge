class Playlists

A singleton interface for manipulating playlists

Methods

  • load ()

    Updates the playlists from the 'database'.

  • save ()

    Saves the changes made to the lists to the 'database'.

  • create (name)

    Creates a new playlist.

    Arguments

    • String name

      The name of the new playlist.

  • remove (playlist)

    Removes a playlist from the list of playlists.

    Arguments

    • Object playlist

      The playlist to remove.

  • getByName (name)

    Gets a playlist by its name.

    Arguments

    • String name

      The name of the playlist.

  • getByShortName (sname)

    Gets a playlist by its short name.

    Arguments

    • String sname

      The short name of the playlist.

  • createShortName (name)

    Creates a short name for a playlist based on its name.

    Arguments

    • String name

      The name of the playlist.

  • getAll ()

    Gets all playlists the user has.

class FoundTrack

A singleton interface to manipulate tracks found by search.

Methods

  • addToPlaylist (The)

    Adds a track to the playlist.

    Arguments

    • DOMElement The

      DOM element to parse the track's properties from.

  • addManyToPlaylist (The)

    Adds multiple tracks to the playlist.

    Arguments

    • Array The

      array of elements to parse the tracks' properties from.

class Track

A class to represent a track in the playlist.

Methods

  • fromElement (elem)

    Gets the data from a DOM element and creates a Track of that data.

    Arguments

    • DOMElement elem

      The DOM element to get the data from.

  • remove ()

    Removes the track from its playlist.

  • move (index)

    Moves the track on the playlist.

    Arguments

    • String|Number index

      Either 'up' or 'down' or an absolute index.

  • update ()

    Updates the track's UI.

  • play ()

    Plays the track.

class Playlist

A singleton interface for manipulating the current playlist.

Methods

  • setName ()

    Sets the name of the playlist and updates the UI accordingly.

  • setDescription (v)

    Sets the description of the playlist and updates the UI accordingly.

    Arguments

    • String v

      The new description.

  • load (sname)

    Loads a playlist based on its short name.

    Arguments

    • String sname

      The short name of the playlist.

  • save ()

    Saves the playlist to the 'database'.

  • add (The, index)

    Adds a track to the playlist and updates the UI accordingly.

    Arguments

    • Track The

      track to add.

    • optional Number index

      The position to add to. If not specified, appends to the list.

  • remove (The)

    Removes a track from the playlist and updates the UI accordingly.

    Arguments

    • Track The

      track to remove.

  • updateButtons ()

    Updates the UI for all the tracks in the playlist.

class Search

A singleton interface for manipulating the search functionality.

Methods

  • search (query)

    Searches for the query and displays the results.

    Arguments

    • String query

      The search query.

  • showResults (tracks)

    Shows the results for a search.

    Arguments

    • Array tracks

      The resulted tracks.

  • hideResults ()

    Hides the results box.

class Player

A static interface for manipulating the player widget.

Methods

  • play (track)

    Starts playing the playlist at a specified track.

    Arguments

    • Track track

      The track to play.

  • initWidget (track)

    Initializes the player widget.

    Arguments

    • Track track

      The track to initially load the widget with.
